# Administrador de tareas

## Introduccion
Esta es una aplicación SPA desarrollada con VUE 3 y Vue Router que permite gestionar tareas personales: agregar, editar, marcar como completadas y eliminarlas, sin actualizar la página. Este ejercicio se enmarca en el trabajo final del módulo del Bootcamp de Front-End de Talento Digital.
